Suggestion Guidelines
  • Provide a clear, descriptive name for the game
  • Include detailed step-by-step instructions
  • Specify the focus areas (e.g., dribbling, teamwork, shooting)
  • Indicate the number of players needed
  • List all required materials
  • Estimate the typical duration
  • Include any variations or modifications
  • Add relevant labels for easy categorization
Markdown Formatting

You can use markdown formatting in the description and variants fields:

  • Bold text: **bold** or __bold__
  • Italic text: *italic* or _italic_
  • Underlined text: <u>underlined</u>
  • Lists: Use - or * for bullet points
  • Numbered lists: Use 1., 2., etc.
